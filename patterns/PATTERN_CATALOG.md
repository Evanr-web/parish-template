# Byzantine Pattern Catalog — Parish Template

All patterns extracted from **public domain** sources (19th century, authors deceased 100+ years).

## Sources
- **Owen Jones** — *The Grammar of Ornament* (1856/1910)
- **Heinrich Dolmetsch** — *Der Ornamentenschatz / Styles of Ornament* (1906)
